## Comment remplacer une image

1. Ajoutez votre photo dans ce dossier (`frontend/public/images/`)
2. Mettez à jour le chemin correspondant dans `frontend/src/pages/LandingPage.jsx`

## Formats acceptés

`.jpg`, `.jpeg`, `.png`, `.webp`
